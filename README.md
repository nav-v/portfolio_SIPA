# Portfolio SIPA

This repository contains my data analysis portfolio built with Jupyter Book.

## Contents

This portfolio includes various data analysis projects and notebooks covering topics such as:
- Birth rates analysis
- Data visualization with Plotly
- Statistical analysis
- And more!

## Building the Book

To build this Jupyter Book locally:

```bash
pip install -r requirements.txt
jupyter-book build --all .
```

The built HTML will be in `_build/html/`.

## Viewing the Portfolio

The portfolio is automatically deployed via GitHub Actions and can be viewed at the deployed site.

## Requirements

See `requirements.txt` for all required Python packages.
